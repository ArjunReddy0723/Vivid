Vivid

A visual testing tool to compare and determine the visual differences between two versions of same web page

Environment setup

Mac OS-Run following commands to setup the environment-

  • Open terminal and copy paste below commands one by one
  • ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
  • Press Enter
  • Enter your NT password
  • Wait for current installation to finish
  • brew install imagemagick
  • brew install phantomjs

Windows- Run following commands to setup the environment-

  • Download http://www.imagemagick.org/download/binaries/ImageMagick-6.9.0-0-Q16-x64-dll.exe
  • Double click to install it. Accept all the default settings

How to build the jar file-

User can build either swing based UI application or a command line jar.

  • Build Swing application-Comment line#103 in Vivid.java and buld the application using maven Command
  • Build command line jar-Comment from line#87 to line#96 and build the application using maven command

How to use Swing application?

  1. Create a directory Vivid under your username on windows it will be C:\Users[your NT user name] on Mac /Users/[your user name]
  2. Download the jar
  3. Copy the jar from Download folder to Vivid folder
  4. Double click the jar file
  5. Provide the url pairs in URLs text field. You can provide multiple url pairs, each pair in a new line. The format of the url is as follows- url=URL1|URL2. Be informed that there is a " | " in between url1 and url2
  6. Select the type of device you want to compare you web pages on. The selectable values are Desktop, Tablet and Mobile
  7. Select the number of threads you want application to create. If you don't change the preselected value then default would be 5. Make it 10 if you have more than 10 url pairs.
  8. Click on Run to run the application. Clicking on Exit will close the application
  9. If you have multiple urls you can create a file having multiple url pairs each starting with new line and upload it using upload urls button.
  10. Refer the below video for quick introduction of the application. http://www.powtoon.com/show/ettk3bdHIAK/how-to-use-ebay-vivid/

How to use CLI?

  1. Create a directory Vivid under your username on windows it will be C:\Users[your NT user name] on Mac /Users/[your user name]
  2. Download the jar
  3. Copy the jar from Download folder to Vivid folder
  4. Download the config.properties file and copy it to Vivid folder
  5. Delete the existing URL and put your URL. Do not change anything else
  6. Open command line
  7. Windows: cd C:\Users[username]\Vivid
    Mac: cd /Users/[username]/Vivid
    Make sure to replace [username] with your username without [ ]
  8. Run the command java -jar vivid-0.0.1-SNAPSHOT-cli.jar -propertyFilePath config.properties
  9. Press Enter
